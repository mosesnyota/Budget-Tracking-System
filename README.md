## About This Application

The application was designed to help an organization in Kenya manage their projects; by recording every expense per project and be able to produce expenses reports against a budget. 

- Create Budget Voteheads/expense line and specify amount allocated for that budget line
- Record expenses against a votehead/budget line
- Produce expenses reports for a given period for the voteheads
- Produce full project reports in PDF
- Help management track budgets for projects

The application is developed using Lavavel Framework.
